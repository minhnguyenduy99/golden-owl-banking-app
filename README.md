# I. Installation

Install NodeJS version *18.19.x*

Install npm packages:

```
npm install
```

Run the application:

```
npm run start:dev
```

The application is now running at [http://localhost:3000]([http://localhost:3000])

To open Swagger, open: [http://localhost:3000/swagger](http://localhost:3000/swagger)
